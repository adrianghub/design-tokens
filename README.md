# @adrianghub/design-tokens

Design tokens for the @adrianghub design system.

## Installation

```bash
npm install @adrianghub/design-tokens
```

## Usage

```js
import { colors } from '@adrianghub/design-tokens';

console.log(colors.primary);
```
